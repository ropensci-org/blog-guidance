C Template - Author file

Use of this template is described in Create or update your author file.

Author file template to be saved as /content/authors/yourfirstname-yourlastname/_index.md as described in Technical Guidelines.

You can hover over the top-right corner of the template to make a copy-paste button appear.

---
name: Author Name
link: website URL or other online presence
twitter: Twitter username
github: GitHub username
gitlab: GitLab username
keybase: Keybase ID
orcid: ORCID ID
---
